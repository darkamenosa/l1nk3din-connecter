# L1nk3dIn Connecter

A simple tool to click connect buttons on Linkedin.

## Getting Started

Create a `.env` file, then put your linkedin username and password there

```
USER_NAME=
PASSWORD=
```

Install node dependencies and start:
```
npm install
npm run start
```

### Prerequisites

- NodeJS 7+

## Development
Run `npm run dev` to see what is going on.
